# This is a simple app where a user can input information and upload a file

# Built using Express & React


# To Start the App
-> download all necessary npm packages by running "npm install" in the command line tool
-> install necessary npm packages in client as well
  (cd client -> npm install)
-> create postgresql database "Parallel" (run createdb Parallel in the terminal if postgres is already installed)
-> "npm run dev" command will start the client and the server
